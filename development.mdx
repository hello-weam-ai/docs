---
title: "Service Access URLs"
description: "Quick reference to access services running in Docker during development"
---

<Info>
  **Services available via Docker Compose**:
  - Frontend, APIs, Databases, Object Storage, and Debugging tools
  - No extra config needed if Docker is running
</Info>

Use this guide as your quick-access panel for URLs and ports during development.

<Steps>

<Step title="Frontend Interface">

```url
http://localhost:3000
```
**Next.js Frontend UI**

</Step>

<Step title="API Services">

- Node.js Backend → [`http://localhost:4050/napi`](http://localhost:4050/napi)  
- Python Gateway (FastAPI) → [`http://localhost:9089/pyapi`](http://localhost:9089/pyapi)

</Step>

<Step title="API Documentation">

- Swagger UI → [`http://localhost:9089/docs`](http://localhost:9089/docs)  
- Redocly → [`http://localhost:9089/redoc`](http://localhost:9089/redoc)  
- Postman → [Postman Collection](https://documenter.getpostman.com/view/36531136/2sB34Zr4fg)

</Step>

<Step title="Vector Database (Qdrant)">

- Qdrant UI → [`http://localhost:6333`](http://localhost:6333)  
- Node Port → [`http://localhost:6335`](http://localhost:6335)

</Step>

<Step title="Developer Tools">

- Mongo Express → [`http://localhost:8081`](http://localhost:8081)  
- Python WDB Debugger → [`http://localhost:1984`](http://localhost:1984)

</Step>

<Step title="Databases">

- MongoDB: Accessible via `localhost:27017` (use Compass or CLI)  
- Redis: Internal access only (via Redis CLI)

</Step>

<Step title="Object Storage (MinIO)">

- MinIO Dashboard → [`http://localhost:9001`](http://localhost:9001)  
- S3-compatible API → [`http://localhost:9000`](http://localhost:9000)

</Step>

</Steps>

<Note>
  You can update ports in the `docker-compose.yml` file if conflicts arise. For custom environments, document overrides in your `.env` or service readmes.
</Note>

## Troubleshooting

<AccordionGroup>

<Accordion title="Service not accessible at localhost">

Ensure:
1. Docker is running and healthy
2. The service is listed in `docker-compose.yml`
3. Ports are not blocked or in use by other applications

</Accordion>

<Accordion title="MinIO or Mongo Express show blank or fail to load">

Clear browser cache or restart Docker. These services can be sensitive to startup order and may require restarting containers individually.

</Accordion>

</AccordionGroup>
