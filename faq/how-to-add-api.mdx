### Currently Supported Providers

Weam currently supports the following providers:

- **Directly integrated:**
  - OpenAI
  - Anthropic
  - Gemini
  - Hugging Face

- **Via OpenRouter:**
  - DeepSeek
  - LLaMA
  - Qwen
  - Grok
