## Ways to Contribute

There are many ways you can contribute to the project:

- **Report Bugs**: Found a bug? Please [open an issue](https://github.com/weam-ai/weam/issues/new?template=bug_report.md).
- **Suggest Features**: Have an idea to improve Weam AI? [Submit a feature request](https://github.com/weam-ai/weam/issues/new?template=feature_request.md).
- **Fix Issues**: Check our [open issues](https://github.com/weam-ai/weam/issues) and help resolve them.
- **Improve Documentation**: Found something unclear or outdated? Feel free to propose improvements.
- **UI/UX Contributions**: Help us improve the user interface and experience.
- **Testing & Feedback**: Try out new features and provide feedback.
- **Translation**: Help make Weam AI accessible in more languages.

## How to Contribute

1. **Fork** the repository.
2. **Create a new branch** for your work:  
   ```bash
   git checkout -b my-feature
   ```
3. **Make your changes** and **commit** them with a meaningful message.
4. **Push to your fork** and open a **pull request**.

> Please ensure your changes follow our code style and guidelines. Pull requests should be linked to an open issue when possible.
