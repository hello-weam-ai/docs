Agents in Weam AI are specialised AI assistants that can be customised for specific tasks and roles. These agents are designed to handle particular types of interactions based on their configured knowledge base.

## Benefits of Using Agents

- **Customised Responses**: Get tailored responses for specific departments or tasks
- **Model Selection**: Choose specific LLM models that best suit your agent's purpose (including Claude 3.5 Haiku Latest and other advanced models)
- **Streamlined Workflows**: Enhance departmental workflows with dedicated AI assistants
- **Focused Support**: Perfect for teams needing specialized AI support for particular functions
- **Cross-Team Access**: Team members can access agents across the platform, allowing use of agents created by other members even if they're not part of that particular brain

### Agent Configuration Options

#### Brain Assignment
Agents can be assigned to different organizational "brains" including:
- **Shared Brains**: For example, Helios, UI/UX, PMs, QA, General Brain, Merchandise Shop, System Designer, HR, Development, Sales, Marketing
- **Private Brains**: For example, Client Success, Social Media, and custom private brains

#### System Prompt Best Practices
- Define the agent's primary role and expertise area
- Specify the tone and communication style
- Include relevant background context and knowledge
- Set clear boundaries for the agent's capabilities

## How Agents Work

### Agent Management

Users can easily switch between regular chat mode and agent-specific conversations. When chatting with an agent, responses are tailored to that agent's specific expertise and training. The platform provides:

- **Agent Overview**: View all available agents with descriptions
- **Search Functionality**: Find specific agents quickly
- **Favorites**: Mark frequently used agents for quick access

## Agent Library

The Agent Library is your central hub for discovering and accessing pre-built specialized AI assistants. Browse through a curated collection of agents designed for various business functions, each optimized with specific system prompts, goals, and AI models.
Key Features:

- Search & Browse: Find agents using keywords or browse the complete collection
- Quick Preview: View agent details, capabilities, and AI model specifications
- Move to Brain: Add agents to your preferred Brains for regular use
- Team Sharing: Access agents across Shared and Private Brains
